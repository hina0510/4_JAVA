Collection Framework
- List : 순서가 있는 데이터의 집합, 데이터의 중복을 허용
         배열은 고정적인 크기를 갖지만 리스트는 가변적인 크기를 가짐

.add("데이터") : 추가
.remove(주소 or "데이터" ) : 삭제
.get(주소) : 번째의 데이터를 가져옴(출력 시 주로 사용)
.clear : 데이터 전체 삭제
.indexOf("데이터") : 데이터의 주소 출력 

- Set : (Hashset)순서가 존재하지 않음, 중복을 허용하지 않음
         크기 기본 16개의 배열, 75% 정도 데이터가 차면 새롭게 16개 늘림

반복자 : 순차적으로 값이 저장되며 제일 앞에는 bof, 제일 끝에는 eof
hasNext : 다음 위치에 값이 있는지 없는지 확인, 값이 있으면 true, 없으면 false
next : 다음 위치로 이동되고 값을 가져온다
