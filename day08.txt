class : 사용자 정의 자료형, 중복되는 값들은 하나의 자료형으로 만들어서 계속 사용
object(객체) : 변수, 클래스의 자료형으로 변수를 만들면 객체로 표현
	     첫 글자가 소문자면 변수(int), 대문자면 객체(String)
	     모든 기능을 가지고 있는 것

public : 외부 또는 내부에서 접근 가능
private : 내부에서만 접근 가능
default : 

method : 기능, 클래스 안에 기능이 있으면 method라고 표현
function(함수) : 기능, 클래스 없이 단독적으로 기능이 있으면 함수

parameter(인자) : main파일에서 class파일에 보내는 값
argument(매개변수) : main파일에서 입력한 값을 class파일에서 받는 변수

class파일에 return에 값이 있으면 값을 돌려주고 메소드를 바로 종료한다
		  값이 없으면 메소드만 종료한다
메소드의 자료형은 return과 일치시켜야 한다
